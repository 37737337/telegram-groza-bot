# Telegram Groza Bot

Бот сохраняет всех пользователей, которые писали в группе,
и по команде /all тегает их.

## Установка

pip install -r requirements.txt
python bot.py

## Важно
- Бот должен быть админом
- Он может тегать только тех, кто писал
- Не злоупотребляйте, Telegram может ограничить спам
